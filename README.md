# Bike Sharing Demand Analysis Using Excel

## Project Overview
This project analyzes bike-sharing demand using Excel, Pivot Tables, Dashboards, VBA Automation, and AI-driven insights.

## Objectives
- Data Cleaning
- Data Merging
- Weather Analysis
- Hourly Demand Analysis
- Dashboard Development
- VBA Automation

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- VBA Macros
- Slicers
- Forecasting

## Key Findings
- Clear weather generated maximum rentals.
- Peak demand occurred around 8 AM.
- Registered users contributed most rentals.
- Rental demand decreased during adverse weather conditions.

## Deliverables
- Excel Workbook (.xlsm)
- Interactive Dashboard
- VBA Automation
- AI Insights
- Summary Report
